
// Two types of detection:
//  1. user agent detection (browser only)
//  2. object detection
//
// User Agent Detection
// --------------------
// base2 creates its own userAgent string for easy pattern matching.
// 1. platform prepended to user agent string 
// 2. bogus MSIE indicators removed 
// 3. spaces and "/" characters removed between words and numbers
//    e.g. MSIE 6.0 -> MSIE6.0
// Detect user agents by passing a pattern to detect() to match against.
// e.g. detect("MSIE5");      // detects IE5.x
// e.g. detect("MSIE[56]");   // detects IE5.x and IE6.x
// e.g. detect("MSIE|opera"); // detects Interent Explorer and opera
// e.g. detect("MSIE.+win");  // detects Interent Explorer on Windows
//
// Object Detection
// ----------------
// Detect objects by passing an expression that can be evalauted. The
// expression should be enclosed in brackets.
// e.g. detect("(java)");
// e.g. detect("(document.getElementsByClassName)");
// Certain special objects are always defined (even if they are null) 
// when evaluating expressions.
// They are:
// * java (if it is enabled)
// * jscript (version number)
//   e.g. detect("(jscript)");
//   e.g. detect("(jscript>=5)");
// * global (the global object. e.g. window in a browser environment)
// * element (an HTML <span> element which is not attached to any document)
//   e.g. detect("(element.addEventListener)");
// * userAgent (the modified base2 userAgent string)

// You can negate the return value of a detect() by prefixing the query
// with a "!" character.
// e.g. detect("!MSIE");
// e.g. detect("!MSIE|opera");
// e.g. detect("!(java)");
